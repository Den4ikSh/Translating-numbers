# About the Project
## What this project is
### Console program for converting numbers between different numeral systems
> [!NOTE]
> ## Purpose
> - Converting Arabic numerals to other numeral systems;
> - Converting various numeral system numbers to desired ones;
> - Viewing specific numeral systems in comparison with Arabic.
> ## Goal
> - Converting numbers between 6 numeral systems (Arabic, Roman, Greek, etc.) via a console interface.
> ## Important
> - `main.py` - the source code of the program.
> - `TranslateNumbers.exe` - a compiled and ready-to-use python file. It only works on Windows.
> - The program is laid out as an example and its functions can be openly used.

> [!TIP]
> ## Features
> - Functional requirements:
> 	- Inputting numbers via console/file;
> 	- User selection of source and target numeral systems;
> 	- Input validation;
> 	- Outputting the result to the terminal.
> - Limitations:
> 	- Roman numerals only up to 3999;
> 	- Support for positive integers only.
> - The program can:
> 	- Accept a number from the user;
> 	- Convert it to the specified system;
> 	- Output the result or an error message;
> 	- In case of an error, provide the user with the option to re-enter the number or return to the conversion menu.

> [x] The project is fully completed and does not require any further modifications (except for possible GUI addition)
